Este é um aplicativo interativo de Sound Pad, criado para estudar e praticar manipulação do DOM em JavaScript. O usuário pode tocar diferentes efeitos sonoros ao clicar em botões ou pressionar teclas de atalho, como a barra de espaço ou Enter. O projeto demonstra como interagir com elementos HTML, adicionar eventos e manipular atributos e classes de forma dinâmica.

Funcionalidades:

Interface com botões para tocar sons de instrumentos.
Atalhos de teclado para cada som (barra de espaço e Enter).
Feedback visual ao tocar um som, destacando a tecla ativada.
Manipulação de elementos <audio> para reprodução dos sons.
Tecnologias Utilizadas:

HTML5
CSS3
JavaScript (DOM)

Este projeto foca no estudo do DOM e sua manipulação com JavaScript:

Seleção de Elementos: Usando document.querySelector e document.querySelectorAll para selecionar os botões e associar sons aos mesmos.
Eventos: Inclusão de eventos de clique (onclick) e de teclado (onkeydown e onkeyup) para interagir com os elementos.
Manipulação de Atributos e Classes: Modificação dinâmica das classes e controle de reprodução de áudio.
